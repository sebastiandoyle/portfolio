# Portfolio

Personal portfolio site for Sebastian Doyle — [sebastiandoyle.vercel.app](https://sebastiandoyle.vercel.app)

## Tech Stack

- **Next.js 16** with App Router
- **React 19**
- **Tailwind CSS 4**
- **Framer Motion** for animations
- **Zustand** for state management
- **TypeScript**

## Themes

Four switchable visual themes:

| Theme | Vibe |
|-------|------|
| **Cosmos** | Dark space with animated star field |
| **Cloud Dancer** | Clean, light, minimal |
| **Midnight** | Deep dark blue |
| **Ember** | Warm dark tones |

## Features

- **Voice Void** — interactive demo at `/void`. Speak a description, watch it materialize as a live React component using OpenAI and Web Speech API
- **Project case studies** — detailed breakdowns of featured projects with architecture diagrams
- **Responsive design** — works across all devices
- **Theme persistence** — selected theme persists across sessions

## Development

```bash
npm install
npm run dev
```

Runs on [localhost:3000](http://localhost:3000).

## Deployment

Deployed on Vercel. Pushes to `master` auto-deploy to production.
